# FTP Uploader

Simple FTP uploader.

## Install

This package is meant to be installed globally.

```sh
npm install ftpup --global
```

## Usage

```
  Usage
    $ ftpup [OPTION]... [SRC] [USERNAME[:PASSWORD]@]HOST[:DEST]

  Options
    -u, --username username
    -p, --password password
    -i, --ignore   ignore pattern
```
